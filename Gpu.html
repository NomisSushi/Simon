GPU

Une GPU (Graphics Processing Unit) est un processeur spécialisé conçu pour gérer les calculs complexes nécessaires pour le rendu d'images, de vidéos et d'animations. Alors qu'un CPU est un processeur généraliste qui gère une large gamme de tâches, une GPU est spécifiquement optimisée pour les calculs liés aux graphismes. Une GPU est généralement composée d'un grand nombre de petits cœurs de traitement qui travaillent ensemble pour effectuer des calculs en parallèle, lui permettant de traiter de grandes quantités de données simultanément. Les GPU sont couramment utilisées dans les jeux vidéo, la modélisation 3D, la recherche scientifique et d'autres applications qui nécessitent un traitement graphique à haute performance.
